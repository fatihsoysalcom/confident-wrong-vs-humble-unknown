# Confident Wrong vs Humble Unknown

This Python script demonstrates the difference between a developer who confidently provides an incorrect answer and one who admits they don't know. It simulates API request processing to highlight the negative consequences of 'confidentially wrong' decisions versus the positive outcome of acknowledging uncertainty.

## Language

`python`

## How to Run

Save the code as `confident_wrong.py` and run it from your terminal using: `python confident_wrong.py`
